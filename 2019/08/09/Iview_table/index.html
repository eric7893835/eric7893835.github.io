iview table做跨页多选

由于iview的table并没有自带的分页功能，只能由Page组件实现分页。实现思想是每次分页就给table的data重新填充数据。

但是做多选的时候，分页就会出现bug，选中之后跳转到其他页，再跳转回来，就发现以前选中的行，选中效果不见了。
